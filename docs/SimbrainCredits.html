Tools

This software uses various Apache tools, and the following libraries: Beanshell, Colt, GlazedLists, JAMA , JFreeChart, JGAP,  Piccolo, OjAlgo, RText and RSyntaxTextArea, SSJ, SwingLabs, and XStream. Some of the tools at Ostermiller.org are also used. Bits of open source code found online or in books are noted in code documentation.

The Simbrain.net website is based on the skeleton framework and uses an unslider.

Funding

The development of Simbrain has been supported by several grants (Hewlett Foundation, UC Merced development grant).
