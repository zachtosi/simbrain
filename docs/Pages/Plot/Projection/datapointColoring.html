Datapoint Coloring


Decay Trail: The current point gets the greatest saturation and it trails off in a decaying saturation after. Decays based on decay increment.

Frequency: As a point is visited more, its activation and hence the saturation of its color representation increases.

Hot Point Mode: When selecting this option, it enables the hot color.

Base Color: The base color to be used for all points besides the current point.

Hot Color: The "hot color" to be used for the current point.

Ceiling: The lower bound for "activation" of a point in frequency and decay trail.

Floor: The upper bound for "activation" of a point in frequency and decay trail. Max is 1.

Decrement Amount: How much to decrement activation at each time step (used in decay trail coloring method).
